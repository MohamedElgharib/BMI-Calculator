# BMI Calculator

## Introduction
This project is a simple Java program that calculates the Body Mass Index (BMI) of a user. 
The program asks the user to enter their weight and height and then calculates the BMI using a mathematical formula. 
It also displays the BMI category such as Underweight, Normal weight, Overweight, or Obese.

## Technologies
- Java
- VS Code
- GitHub
- Markdown

## Features
- User inputs weight in kilograms
- User inputs height in meters
- Calculates BMI using the BMI formula
- Displays BMI value
- Shows BMI category based on the result

## BMI Formula
BMI = weight / (height × height)

## How to Run
1. Download the project files
2. Open the project in VS Code or any Java IDE
